# Link to SDK

Using CocoaPods

## Repos

Specify the following sources at the beginning of your `Podfile`:

```ruby
source 'https://github.com/cocoapods/specs' 
source 'https://github.com/jivochat/cocoapods-repo'
```

## Dependency

Add SDK as dependency into your `Podfile`:
```ruby
pod 'JivoSDK'
```
